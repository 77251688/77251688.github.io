🤔开发插件

前言​

本框架采用TypeScript或JavaScript开发 推荐使用TypeScript开发

采用TS开发编辑器会有代码提示,类型检查等等好用的功能...

开始开发​

安装TypeScript​

npm i -g typescript

检查ts版本

tsc -v

能正常输出版本号即可

可能你们输入tsc-v会报错,这个问题很简单上网搜一下就解决了

插件模板​

TS​

TypeScript
import { Client } from "@cummins/oicq";
function listener() {
    /** 编写代码 */
}
function enable(bot: Client) {
    bot.on("message", listener);
}
function disable(bot: Client) {
    bot.off("message", listener);
}
export { enable, disable };

JS​

JavaScript
"use strict";
function listener(e) {
    /** 编写代码 */
}
function enable(bot) {
    bot.on("message", listener);
}
function disable(bot) {
    bot.off("message", listener);
}
module.exports = { enable, disable };
